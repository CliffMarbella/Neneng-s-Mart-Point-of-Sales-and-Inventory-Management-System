
Install Point Of Sales System
1. Run setup.exe
2. Follow on screen instruction
3. Finish

Install Mysql Database
1. Run mysql-installer-web-community-5.6.41.0
2. Make sure there is an internet connection
3. Wait for packages to download
4. Follow on screen instruction
5. Open Mysql Workbench
6. Open Connection
7. On the tool window, Select Server
8. Click Data Import
9. Select Self Containing File
10. Select Database.sql on Database folder of this CD
11. Click Start Import
12. Done!
